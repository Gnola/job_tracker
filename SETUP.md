## Set Up Project

In **root directory**

```bash
npx create-react-app client
```

### Install Dependencies

In **/client directory**

```bash
npm install @mui/material @emotion/react @emotion/styled
```

---

### Run Locally

| Location | Command      | Runs         | URL                     | Access |
| :------- | :----------- | :----------- | :---------------------- | :----- |
| /client  |  `npm start` | **Frontend** | *http::/localhost/3000* | Site   |
